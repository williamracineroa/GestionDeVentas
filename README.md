# 🛒 Sistema de Gestión de Ventas en Python

## 📌 Descripción

Este proyecto es un programa básico en **Python** que simula un sistema sencillo de gestión de ventas.  
Permite:

- Registrar el nombre del cliente
- Ingresar el precio unitario de un producto
- Ingresar la cantidad de productos
- Aplicar un descuento del **10%** si el cliente cuenta con membresía VIP
- Mostrar un resumen detallado de la venta

Es ideal como práctica para principiantes que están aprendiendo:

- Entrada de datos (`input`)
- Conversión de tipos (`float`, `int`)
- Condicionales (`if`)
- Operaciones matemáticas
- Formato de strings con `f-strings`
